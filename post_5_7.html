ROS Workshop and Preliminary Report

Posted by Avinash Sen on April 04, 2020

Attended a ROS(Robot Operating System) and Python Workshop which will be very useful for the development of my project.

In this workshop each participant would have to work on the tutorials of Turtle Sim Robot in ROS. Each ros topic representing each functions which would simulate a turtle according to the code written by the participant.

This workshop was also helpful to increase my skills using the GitHub platform.

Besides the workshop this week was also dedicated to do some initial research and to write the preliminary report. This was important for me, not only, to introduce myself to the process of bin-picking and the projects that have already been developed in this field, but also, to organize my thoughts in tasks to come in this project.
